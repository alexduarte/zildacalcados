Prazos de Entrega

Objetivo

    Definir os prazos de entrega das transportadoras e serviços dos Correios quando utilizando a forma de cálculo offline.

Capital

    Informe o prazo de entrega dos serviços dos Correios para a Capital de cada Estado.

Interior

    Informe o prazo de entrega dos serviços dos Correios para as cidades do Interior de cada Estado.

Botão Salvar

    Clique no botão para validar as alterações efetuadas.

Observaçao

    As opções inativas somente estão disponíveis na versão comercial do produto.
